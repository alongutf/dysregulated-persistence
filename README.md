# dysregulation_in-antibiotic-tolerance-persistence
All code and processed data for reproducing the results presented in "Genome-wide Dysregulation in Antibiotic Tolerance and Persistence". Contains the original code to produce the \textit{Generalized Marchenko-Pastur} distribution and to fit the model to experimental data. Analysis includes data filtering and processing, calculation of correlation matrices and eigenvalues and further scRNA-seq analysis.
